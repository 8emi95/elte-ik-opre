
EXECUTE SERVER AND CLIENT
-------------------------

run server and client side in separated window as separated process. 
Client and server communicate via named pipe.

SERVER SIDE
-----------

you can use main executable in server mode

> ./mlsz server


CLIENT SIDE
-----------

you can use main executable in client mode

./mlsz client <client_name>

> ./mlsz client budapest-i


client automaticly connects to server and gets all questions and replies answers for each questions
